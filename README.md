# Data Analytics Fundamentals

## Overview
This repository contains coursework, projects, and practice exercises completed for a Data Analytics course. The class introduces the lifecycle of data analysis and provides hands-on experience with statistical testing, predictive modeling, and data interpretation using modern analytics tools.

The goal of this repository is to demonstrate my ability to analyze data, apply statistical methods, and use data analysis tools to solve practical problems across multiple domains.

## Course Description
This course serves as an entry-level class in a data analytics learning sequence designed to prepare careers in data analytics and data-driven decision making.

The course covers the complete lifecycle of data analysis including:

- Formulating research questions
- Data collection
- Data analysis
- Interpreting results
- Making data-driven decisions

Students apply both **parametric and non-parametric statistical tests** and build **predictive models** using real-world datasets from domains such as:

- Information Technology
- Healthcare
- Education
- Manufacturing
- Natural Sciences

Commonly used analytics tools include:

- Python
- R
- SPSS
- Tableau

## Topics Covered

- Data analysis lifecycle
- Research question formulation
- Data collection and preprocessing
- Statistical significance testing
- Parametric and non-parametric tests
- Linear regression
- Logistic regression
- Predictive data modeling
- Data interpretation and decision making
- Data visualization

## Tools & Technologies

- R
- SPSS
- Tableau
- Statistical analysis techniques
- Data visualization tools

## Learning Outcomes
After completing this course, I developed the ability to:

- Apply the full lifecycle of data analysis
- Use statistical tests to analyze datasets and evaluate hypotheses
- Build predictive models using regression techniques
- Manipulate and analyze datasets using modern analytics tools
- Interpret results and communicate data-driven insights
